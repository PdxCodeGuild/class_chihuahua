# The Hub

---
I will be using API's to call information from varoius sources.

The flow of the site will be as follows 

TheHub

2. Sports

        * MLB
        * NFL
        * College Football
        * College Basketball
    * News 
        * Breaking News
        * Trades
        * Draft Watch
        * Records
3. News
    * Local
    * National
    * Weather
